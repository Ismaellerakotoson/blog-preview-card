### What I learned

During this project, I learned how to:

- Use Sass variables to improve maintainability.
- Import and use a local variable font with `@font-face`.
- Create responsive layouts using modern CSS functions such as `min()`.
- Implement hover effects and smooth transitions.
- Structure styles more efficiently using Sass nesting.

Example of a Sass variable used in this project:

```scss
$primary-color: hsl(47, 88%, 63%);

button {
  background-color: $primary-color;
}
```
